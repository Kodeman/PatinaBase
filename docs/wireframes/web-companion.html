Overview

Executive Summary

Translating mobile's conversational navigation to professional web workflows

The Challenge

The Patina mobile app pioneered "Navigation as Conversation" — replacing traditional tab bars with the Strata Companion. For the web portals serving designers and manufacturers, we face a different challenge: how do we carry this philosophy into professional workflows where efficiency matters, multi-tasking is common, and users expect certain conventions?

The Solution

The Web Companion sits alongside traditional navigation rather than replacing it. Think of it as a knowledgeable colleague always present in the room — never blocking your work, but ready with context-aware suggestions, quick actions, and natural language navigation when you need it. The static sidebar handles spatial memory; the Companion handles intent and insight.

Section One

Design Philosophy

From mobile's navigation-as-conversation to web's companion-as-colleague

Core Principles

How the mobile Companion philosophy adapts to professional web contexts

Principle Mobile App Web Portal
Always Present, Never Intrusive Bottom sheet that breathes quietly Right panel that stays visible but calm
Context Over Configuration Knows current screen, viewing piece Knows project, client, product, and task
Intent Over Instruction Say what you want, Patina navigates Type or click — both paths available
Graceful Uncertainty Clarifying questions in conversation Suggestions with "Did you mean...?"
Navigation Paradigm Companion replaces nav entirely Companion augments traditional nav
💡

The Key Insight

Professional users need spatial memory — they want to know "Projects is second in the sidebar." But they also want shortcuts when they know exactly what they need. The Web Companion offers both: familiar landmarks plus conversational shortcuts.

What We're Keeping

The Strata Mark visual identity. Context-aware suggestions. Natural language understanding. The warm, unhurried voice. Quick action chips. The principle that technology should feel like a thoughtful colleague.

What We're Adapting

Navigation lives in a sidebar, not the Companion. The Companion is a persistent panel, not a pull-up sheet. Keyboard shortcuts for power users. Multi-panel layouts for complex workflows. Desktop-appropriate interaction patterns.
